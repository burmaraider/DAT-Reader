# DAT-Reader
Lithtech DAT World Viewer


## **Overview**
* It can display a .DAT file from AVP2 and NOLF
* It has no UV support
* Ability to hide and unhide the BSP, blockers, and volumes.
* Realtime light preview with shadows
* Object placement
